Snake Game!

Score: 0

Use the arrow keys on your keyboard, move the snake around the board and eat the red apples! Each time you eat an apple, your snake will grow, and your score will go up by 1. However, the game ends if you hit a wall or run into yourself! The snake will also speed up each time it eats an apple. Good luck!
Currently, this game is only programmed to respond to keyboard arrows. Unfortunately, if you are using a touchscreen or other device without keyboard arrows, it will not work. Womp womp.
